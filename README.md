# react-text-editor

> Simple and full customizable rich text editor in react powered by font awesome and bootstrap.


## Usage

```jsx
import React, { Component } from 'react'

import Editor from 'react-text-editor'
import './styles.scss'

class Example extends Component {
  render() {
    return <Editor />
  }
}
```
